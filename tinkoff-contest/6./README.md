### Формат вывода

Для каждого запроса типа ?.

##### Пример 1

| Ввод          |
| ------------- |
| 6 3           |
| 2 4 6 8 10 12 |
| ? 2 5 3 0     |
| + 2 3 6       |
| ? 2 5 3 2     |

| Вывод |
| ----- |
| 10    |
| 11    |
